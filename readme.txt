Средство обнаружения honeypot.

	Обнаружение строится на задержках сетевых пакетов. Применяются задержки для ответов на TCP и ICMP пакеты. 

	Сеть, в которой находится машина может быть просканирована с помощью функции analize_network(). Применяется библиотека nmap для анализа как хостов, представленных в сети, так и прослушиваемых TCP-портов для них. ROC-кривые и confusion matrix могут быть построены с помощью функции  test().
	
	Для отправки ICMP-пакетов и замера задержек для них применяется библиотека ping3, для аналогичных действий с TCP-пакетами применяется билиотека tcp_latency. 
	
	В качестве классификатора применяется sklearn.gaussian_process.GaussianProcessClassifier (класс Classifier определен в файле clf.py). Для сохранения результатов сканирования была реализована база данных (файл database.py).
	
	Результаты работы программы (база данных, обученные модели, распределения полученных пакетов, построенные ROC-кривые и confuxion matrix) представлены в директории results.
